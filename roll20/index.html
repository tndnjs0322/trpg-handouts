로그 복사-붙여넣기

Roll20의 채팅 기록을 붙여넣으면 편집 기능이 활성화됩니다.

  • Roll20의 채팅 기록 페이지로 이동합니다 (https://app.roll20.net/campaigns/chatarchive/숫자).
  • (선택) Show on One Page 버튼으로 전체 로그를 표시합니다.
  • Ctrl+A → 마우스 오른쪽 버튼 → 복사로 채팅 내용만 복사합니다.
  • 위 입력란에 붙여넣고, 편집 시작을 누르면 에디터가 열립니다.

⚠️ 확장 프로그램이 변환한 채팅 로그는 예상치 못한 결과가 나올 수 있습니다. 가능하면 끄고 복사해주세요.

공지사항

오류/요청은 언제든 알려주세요. 동작은 안내된 흐름을 따릅니다.
